Contributing ​

Development ​

To contribute or make adjustments:

  1. Clone the repository and install dependencies:
bash
git clone https://github.com/robuust/vue-components.git
cd vue-components
yarn install
  1. Run the development server:
bash
yarn dev
bash
yarn docs:dev
  1. Build for production:
bash
yarn build
bash
yarn build:docs
